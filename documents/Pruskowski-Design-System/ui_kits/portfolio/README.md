# Portfolio Website UI Kit

**Product:** Michael Pruskowski personal portfolio — [4n3n2n1.com](https://4n3n2n1.com)  
**Design width:** 1280px desktop  
**Source:** Local codebase at `4n3n2n1.com/`

## Screens

- `index.html` — Full interactive portfolio (Homepage → Work → Contact click-through)

## Components

- `Nav.jsx` — Fixed navigation with logo, social icons, collapsible menu
- `Hero.jsx` — Large Cormorant Garamond headline with italic red accent + footer cells
- `SectionHeader.jsx` — Full-width red band with mono section tags
- `Marquee.jsx` — Scrolling text strips (capability tags + brand names)
- `PillarGrid.jsx` — 2×2 capability cards with Material icons
- `ServiceList.jsx` — 4-column service cards
- `QuoteSection.jsx` — Centered italic pull quote with parallax scale/opacity
- `CTABand.jsx` — Full-bleed photo + overlay with large Cormorant heading
- `ContactSection.jsx` — Info rows + transparent email form
- `Footer.jsx` — Footer bar with logo variants

## Usage

Open `index.html` to see a clickable prototype of the homepage. Nav links scroll to sections.
